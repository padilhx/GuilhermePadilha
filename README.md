#  Guilherme Padilha - Portfólio

Portfólio pessoal desenvolvido para apresentar meus projetos, habilidades e evolução profissional como Desenvolvedor Fullstack.

---

## 🛠️ Tecnologias Utilizadas

* React
* TypeScript
* Vite
* Tailwind CSS
* Material UI
* Radix UI
* React Router

---

## 📌 Funcionalidades

* Apresentação profissional
* Exibição de projetos
* Seção de habilidades
* Layout responsivo
* Interface moderna e intuitiva
* Navegação fluida

---

## 🎯 Objetivo

Este portfólio representa minha evolução profissional e técnica nos últimos anos, reunindo projetos, experiências e conhecimentos adquiridos durante minha trajetória na área de tecnologia.

Com o crescimento dos estudos, desenvolvimento de novos projetos e definição mais clara dos meus objetivos profissionais, surgiu também a necessidade de criar uma nova apresentação profissional mais moderna, organizada e alinhada ao meu momento atual como desenvolvedor e analista.

O projeto foi desenvolvido com foco em performance, responsividade e experiência do usuário, funcionando também como uma vitrine das tecnologias e práticas que venho utilizando no meu dia a dia.
